# Lien — Solana Lending Hooks (VS Code)

Visualise Compositions as knot diagrams, run simulations from your editor, and print a deploy plan.

## Commands

- `Lien: Open Hook Designer` — opens a webview with the six standard hooks rendered as ropes on a workbench.
- `Lien: Simulate Current Composition` — runs the deterministic simulator and prints the report to the Lien output channel.
- `Lien: Show Deploy Plan` — prints the Anchor deploy steps for the configured cluster.

## Settings

- `lien.defaultPool` — default pool label for simulations.
- `lien.cluster` — `localnet` / `devnet` / `mainnet` (used only by the deploy plan).
